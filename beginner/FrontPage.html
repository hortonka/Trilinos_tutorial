Trilinos Hands-On Tutorial

Reference and tutorial material:

  1. The Trilinos Project relies heavily upon its website.
  2. Each Trilinos package has Doxygen documentation.
  3. A lot of information can be found on the Trilinos "Getting Started" page.
  4. Trilinos has a tutorial document that you can find on the website.
  5. Several previous Trilinos User Group meetings included a video-recorded tutorial that uses the website.

Download and build the examples on your machine

  1. Download Trilinos
  2. Download and install CMake (if not already done)
  3. Make sure you have the BLAS and LAPACK installed
  4. Build and install Trilinos
  5. Build your program against Trilinos

Trilinos Package Examples

  1. Learn how to create and use Tpetra dense and sparse linear algebra objects.
    • Lesson 1: "Hello world!" Learn different ways to initialize MPI (or not) and pass an MPI communicator to Tpetra.
    • Lesson 2: Learn how to make a Tpetra vector, given a communicator from Lesson 1.
    • Lesson 3: Learn how to implement a simple numerical algorithm (the power method) using Tpetra sparse matrices and vectors.
    • Lesson 4: Learn different ways to construct a Tpetra sparse matrix.
    • Lesson 5: Learn how to migrate data in a Tpetra object between two different parallel data distributions.
  2. Learn how to solve linear systems using the Belos package of iterative linear solvers, and the Ifpack2 package of preconditioners
    • Create an Ifpack2 preconditioner
    • Solve a linear system using Belos and Ifpack2
  3. Learn how to use Epetra dense and sparse linear algebra objects.
    • Beginner: Construct some vectors and perform basic operations.
    • Intermediate: Find the dominant eigenpair of a matrix using the power method.
  4. Generate test linear systems using the Galeri package.
    • Generate a matrix, discretized 2D Laplacian on a Cartesian grid.
    • Try generating matrices for some different operators, the list of supported operators is here.
  5. Create an algebraic preconditioner using the Ifpack package.
    • Create a preconditioner using the Ifpack preconditioner factory.
    • Try generating different preconditioners, the list of supported parameters for the factory is here.
  6. Solve a linear system using the Ifpack and AztecOO packages.
    • Solve a linear system _Ax=b_ using Ifpack and AztecOO
  7. Solve a linear system using the ML and AztecOO packages.
    • Solve a linear system _Ax=b_ using ML(black-box) and AztecOO
    • Solve a linear system _Ax=b_ using ML and AztecOO, but setting some multigrid options
  8. Solve a linear system using the Ifpack and Belos packages.
    • Solve a linear system _Ax=b_ using Ifpack and Belos
  9. Compute some eigenpairs using the Anasazi package.
    • Compute the smallest eigenpairs of a discretized 2D Laplacian using the block Davidson solver.
    • Compute the smallest eigenpairs of a discretized 2D Laplacian using the LOBPCG solver.
    • Compute the dominant singular values of a 2D discretization of a simple kernel using the block Krylov-Schur solver.
  10. Solve a nonlinear system using the NOX package.
    • Solve a simple nonlinear system using NOX.
    • Solve a simple nonlinear system using NOX.
  11. PyTrilinos tutorial materials.
    • PyTrilinos tutorial slides
    • Solve a simple 2D Laplace problem using PyTrilinos.
  12. Learn how to use utilities in the Teuchos package.
    • Build a parameter list (used to pass parameters to all Trilinos packages).
    • Build a reference-counted pointer (used to eliminate memory leak issues in most Trilinos packages).
    • Build a command-line parser (tool for changing runtime behavior of program, providing documentation for options).
    • Build a time monitor (tool for timing individual methods or operations in a program).
